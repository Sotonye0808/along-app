# Project AI Context

> **Overview:** Along is a social route-sharing platform where users can share, discover, and interact with travel routes and destinations. Built with Next.js 15+ (App Router) and TypeScript, it features a full-stack architecture with Prisma/PostgreSQL for data persistence, Cloudinary for media management, Redis (Upstash) for caching, and JWT-based authentication. The project uses Tailwind CSS and Ant Design for UI, and supports PWA features including service workers and push notifications.
