# Blog Config

**Path:** `app/lib/config/blog.ts`

**Purpose:** Defines blog layout configuration (`postsPerPage`, `featuredCount`) and `DEFAULT_BLOG_CATEGORIES` array (All Posts, Travel Tips, Platform Updates, Community, Route Guides).

**Key exports:** `BlogCategory`/`BlogLayoutConfig` (types), `DEFAULT_BLOG_CATEGORIES`, `BLOG_LAYOUT_CONFIG`
